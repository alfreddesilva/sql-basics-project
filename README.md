# SQL Basics Project — Sales Mini Report

## 📊 Overview
This project demonstrates fundamental SQL skills using BigQuery.

## 🧠 Skills Applied
- SELECT, FROM, WHERE
- ORDER BY, LIMIT
- DISTINCT
- Column & Table Aliases
- SQL Comments
- Logical Query Execution Order

## 📁 Dataset
This project uses a simulated dataset created directly in SQL using a CREATE TABLE statement.

## 📈 Analysis
- Top sales transactions
- Unique customers
- City-based filtering
- High-value transactions
